# Автосалон (CarShop)

Проект по разработке веб-сайта для автосалона.

## Структура проекта:
- Milestone 1: Планирование и сбор ресурсов.
- Milestone 2: Разработка интерфейса и верстка.
- Milestone 3: Тестирование, исправление ошибок и документация.
